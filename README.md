# electric-power

# 电力运检知识图谱

项目路径：
- ac-code 属性抽取代码及数据
- entities-base 实体库
- filters 过滤器
- filters-data 过滤器所使用的相关资源 
- ner-data 命名实体识别数据
- ner-code 命名实体识别最新代码
- rc-code 关系抽取代码及数据
- .gitignore git忽略文件 

本项目为电力运检知识图谱-知识抽取算法项目代码。  
 
本项目后续的应用实例项目（皮皮电力运检知识图谱管理系统）的相关地址如下： 
- [前端](https://github.com/lionztt/power-kg-web) 
- [后端](https://github.com/lionztt/power-kg-node)
